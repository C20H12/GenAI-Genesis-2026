CA Certificate Installation

To allow the proxy to intercept HTTPS traffic without browser warnings, you must install and trust the root CA certificate.

Download ca.crt

Installation Instructions

  1. Linux (Ubuntu/Debian) System Wide
    • Copy the certificate: sudo cp ca.crt /usr/local/share/ca-certificates/
    • Update certs: sudo update-ca-certificates
  2. Firefox (Any OS)
    • Open Settings > Privacy & Security
    • Scroll down to "Certificates" and click "View Certificates"
    • In the "Authorities" tab, click "Import..."
    • Select ca.crt and check "Trust this CA to identify websites"
  3. Chrome / Edge / Chromium (Linux/Windows)
    • Open Settings > Privacy and security > Security
    • Click "Manage certificates"
    • Go to the "Authorities" tab and click "Import"
    • Select ca.crt and check "Trust this certificate for identifying websites"
  4. macOS
    • Double-click the downloaded ca.crt file to open Keychain Access.
    • Add it to the "System" keychain.
    • Double-click the newly added certificate, expand "Trust", and change "When using this certificate" to "Always Trust".
